### 3. [coding] Fix missing ARIA labels
**Files:** All interactive components

- **TextField** (`packages/ui/src/TextField.tsx`): Ensure aria-describedby for errors

### 4. [coding] Verify keyboard navigation
**Files:** All routes, `packages/ui/src/components.css`

- Tab order: visible focus ring on all interactive elements
- Escape key: closes dialogs, dropdowns
- Enter/Space: activate buttons, links
- Arrow keys: navigate lists, dropdowns
- Focus trap: modals, dialogs

### 5. [coding] Add skip links for main content
**File:** `apps/web/src/App.tsx` or `apps/web/src/components/AppLayout.tsx`

- Ensure "Skip to content" link works (already exists in CSS)

### 6. [coding] Verify contrast ratios
**Tool:** axe-core audit

- Ensure all text meets WCAG AA (4.5:1 for normal, 3:1 for large)
- Focus rings meet 3:1 against background
- Interactive elements meet 3:1 against adjacent colors
