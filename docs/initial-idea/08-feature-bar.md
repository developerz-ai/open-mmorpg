# 08 — Feature Bar (match/exceed current WoW baseline)

## Baseline to match (WoW Midnight-tier)
- Player housing: 3D placement tools, customization, shared neighborhoods.
- Open-world dynamic encounter system (Prey-equivalent).
- Solo/small-group scalable endgame (Delves-equivalent).
- Expanded talent trees (class + hero-tree + specialization layers).
- Social hub city · dynamic world events · PvP battlegrounds.
- QoL: cooldown manager, personal resource HUD, transmog, accessibility options.

## Where we exceed (structural advantages)
1. **No realm queues** — autoscaled shards vs. fixed realm caps.
2. **True seamless zones** from day 1 — no loading screens, not retrofitted.
3. **Clean server-authoritative housing/social** on a modern core.
4. **First-class modding** (manifest/plugin) vs. addon-only hacks.
5. **Open asset formats** (glTF/heightmap) vs. closed WMO/ADT — any tool works.
6. **AI companion system via MCP** — no retail equivalent.
7. **Horizontal scale to 1M+** by default — not bolted on.
8. **Reactive living world** — GTA-inspired NPC/faction AI, weather, world social feed ([09](09-gta6-inspiration.md)).

## Faction model
- Ship **2-faction** reference impl (Alliance/Horde-equivalent, fully original names/lore).
- Faction count is **data-driven** — operators add factions #3+ with no core changes ([06](06-modding-and-extensibility.md)).
